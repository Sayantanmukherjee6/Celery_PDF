**#Objective**

Testing Celery based Task Queue Performance.

**#Required Python Files**
	* **no_c_gen3.py** = Code without Celery
	* **c_gen3.py** = Code without Celery


**#Details**
	*Details about running the code is written in code docstring.
	*Install Rabbitmq - **sudo apt-get install rabbitmq-server**.
	*Create a python3 virtualenv.
	*Install project requirements inside virtualenv - **pip install -r req.txt**.
	*Check docstring of **no_c_gen3.py** and **C_gen3.py** for running the code and checking performance.
